Introducing Elle - a revolutionary new wearable application for the visually impaired.

What is Elle?


With Elle, users can easily type and read notes using a braille keyboard, or by feeling the vibrations of morse code or braille code on their skin.

Elle's braille keyboard allows users to type and read messages quickly and efficiently, using a tactile interface that is intuitive and easy to use. The app can also convert text to morse code or braille code, allowing users to feel the vibrations of the code on their skin and read messages that way.

Whether you are blind or have low vision, Elle makes it easy to communicate and stay organized on the go. With Elle, you can take notes, record messages using echo dot, and stay connected with the world around you - all without the need for a lit up screen.

Try Elle today and experience the power of touch and vibration for yourself. With Elle, the possibilities are endless.





What devices are Elle available on?

Elle is available for use on both WearOs based watches and Echo based devices, and it offers a combination of visual and voice-based interfaces to make it accessible to a wide range of users.

The WearOs watch version of the app features a visual interface that allows users to easily save and read notes on their wrist. The Echo Dot version of the app, on the other hand, uses a voice-based interface that allows users to save and read notes using only their voice.

Whether you're on the go or at home, the Elle app makes it easy for visually impaired individuals to stay organized and keep track of their notes. With its seamless integration with WearOs watches and Echo Dot devices, the Elle app is the perfect tool for anyone who wants to take control of their notes and stay organized no matter where they are.



Elle WearOs app functionalities

Elle app homescreen

  • Features three buttons
    • Type using Braille keyboard
    • Read notes using Braille vibrations
    • Read notes using Morse vibrations

Braille keyboard

  • Features 6 buttons representing dots of a Braille layout
  • Type out the raised dots in any order
  • Long press button 6 to register the letter
  • Long press button 3 to register a space
  • Long press button 2 to send the note recorded to the cloud
  • Swipe left to right to exit to the homescreen

Read notes using Braille - sentence screen

  • This screen displays the notes recorded from the cloud
  • Use the - symbol to navigate to the previous sentence
  • Use the + symbol to navigate to the next sentence
  • Upon clicking the sentence, you'll be taken to the letter screen
  • Swipe left to right to exit to the homescreen

Read notes using Braille - letter screen

  • This screen displays the letters of the sentence chosen
  • Use the - symbol to navigate to the previous letter
  • Use the + symbol to navigate to the next letter
  • Upon clicking the letter, the watch will vibrate in a braille sequence - The vibrations will progress row wise and if there is a raised dot there will be a long vibration else there will be a short vibration (total of 6 vibrations per letter)
  • Swipe left to right to exit to the sentence screen

Read notes using Morse - sentence screen

  • This screen displays the notes recorded from the cloud
  • Use the - symbol to navigate to the previous sentence
  • Use the + symbol to navigate to the next sentence
  • Upon clicking the sentence, you'll be taken to the letter screen
  • Swipe left to right to exit to the homescreen

Read notes using Morse - letter screen

  • This screen displays the letters of the sentence chosen
  • Use the - symbol to navigate to the previous letter
  • Use the + symbol to navigate to the next letter
  • Upon clicking the letter, the watch will vibrate in a morse sequence - If there is a dash there will be a long vibration else there will be a short vibration
  • Swipe left to right to exit to the sentence screen
